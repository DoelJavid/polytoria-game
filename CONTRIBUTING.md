# Contributing to Polytoria

Thank you for your interest in contributing to Polytoria! We welcome contributions from the community to help improve our platform.

The full contributor guide can be found in our Documentation:

- Getting started: [https://v2docs.polytoria.com/contributing/](https://v2docs.polytoria.com/contributing/)
- Software setup: [https://v2docs.polytoria.com/contributing/software/prerequisites/](https://v2docs.polytoria.com/contributing/software/prerequisites/)
- PR guidelines: [https://v2docs.polytoria.com/contributing/software/contributor/pull-requests/](https://v2docs.polytoria.com/contributing/software/contributor/pull-requests/)
- DCO requirements: [https://v2docs.polytoria.com/contributing/software/contributor/dco/](https://v2docs.polytoria.com/contributing/software/contributor/dco/)

## Quick checklist

Before opening a PR, make sure to:

- Search for an existing issue or discussion
- Keep the PR focused on a single change
- Sign off your commits with DCO
- Format the code (`dotnet format`)
- Build and test your changes locally

For setup help and early contributor discussion, you can join the Contributors Discord server: [https://discord.gg/HUgEE9FhSU](https://discord.gg/HUgEE9FhSU)
